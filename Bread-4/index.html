紅蟳米糕
滿滿薑味的紅蟳米糕，先炒再蒸，米糕Q軟好吃，鮮味十足！整隻螃蟹一起蒸煮，讓口感和視覺都更添華麗感！

食材

  • 糯米2杯
  • 肉蟹1隻
  • 蟹肉(買不到的話可省略)30g
  • 薑1塊
  • 紅蔥頭5顆
  • 小香菇10朵
  • 蝦米10隻
  • 油蔥酥少許

調味料

  • 麻油4湯匙
  • 豬油或沙拉油2湯匙
  • 鹽1 1/2茶匙
  • 糖10g
  • 白胡椒粉半茶匙
  • 香菇水(泡香菇的水)2杯
  • 太白粉(螃蟹用)少許
  • 米酒少許

...

糯米洗淨後泡水2個小時；香菇洗淨冷水泡軟；蝦米洗淨冷水泡軟；螃蟹買回家後，略微沖洗後泡入鹽水中約1小時，能讓螃蟹把肺清乾淨。

...

薑切薄片；紅蔥頭切片；香菇切細絲；蝦米略切；泡香菇的水請留下來炒米；

...

接著處理螃蟹，用刷子將螃蟹殼都刷洗乾淨，接著拿尖刀從蟹心刺入，便可以除去綁繩，刷洗蟹腳和螯，並剪去腹部和蟹腸，扳開蟹殼，減去蟹殼上的蟹嘴，再除去螃蟹身上兩邊的蟹腮和正中間的蟹心，用流動水清清的衝洗乾靜即可。

...

蟹鉗蓋上布，輕敲到也裂痕，接著再有蟹黃和蟹肉的地方，灑上少許的太白粉。

...

鍋燒熱，放入麻油和豬油，下薑片小火慢煸到薑片捲起，便可先將薑片撈出；

...

下螃蟹略煎，有灑太白粉的地方朝下，煎定型即可，不需煎熟，定型後即可起鍋備用；

...

下紅蔥頭片炒香炒到週圍有點開始變色；

...

放入蝦米和香菇拌炒，

...

下蟹肉，少許鹽和白胡椒粉略炒後，用筷子挑出蟹肉備用；

...

倒入泡過的糯米拌炒；

...

倒入一杯半的香菇水，鹽、糖及1/2茶匙的白胡椒粉小火拌炒,

...

慢慢的炒到湯汁收乾，並試一下米的硬度，應該是軟Q未全熟的口感，可試情況再倒入預留的1/2的香菇水或水，一樣是拌炒到收乾即可，也請順便試味道進行調整。

...

大蒸鍋此時可以開始將水煮滾，容器的竹製蒸籠裡鋪上蒸籠紙或布，略微用水噴濕，鋪上炒好的糯米飯，鋪平後用飯匙略微切一些凹痕.方便均勻受熱；放入大蒸鍋中,大火蒸15分鐘。

...

15分鐘後打開,檢查是否糯米已蒸熟,試出如有夾生則請在表面噴些水後再蒸10-15分鐘；女漢子這次米炒的夠，所以蒸第一個15分鐘後就已經差不多全熟了；確定糯米飯沒有夾生之後就可以排入之前炒好的蟹肉和薑片，中間再擺上少許的紅蔥酥增加香氣。

...

接著擺上螃蟹的身體和蟹螯；

...

最後把蟹殼蓋在蟹身上，灑一點點米酒後，蓋鍋蓋蒸15分鐘即可。如用的是花蟹則再蒸10-12分鐘即可。
